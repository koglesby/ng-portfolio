Hello! Welcome to my portfolio site.

I'm Full Stack Web Developer with a year of proffessional experience building, testing, and maintaining web applications for clients. I like learning new tools and tech, and simply making code work the best way I can, whether that involves changing a layout, building components, testing, writing Backend logic, or anything else that enhances, fixes or improves the application and development process.

I wrote my first lines of code a couple years after graduating from UC Davis with my B.A. in Film Studies. By that time I had learned to use several of Adobe's design and video applications (Photoshop, Illustrator, Premiere, After Effects), but I wanted to learn to create somthing more functional.

So, I enrolled in a coding bootcamp and started building Full Stack applications with Ruby on Rails. I studied computer science fundamentals, data structures, and worked on an Agile team of five student developers to build a functional chess game application. I also got an introduction to JavaScript, and after completing the bootcamp, I became interested in all the ways modern JavaScript could be used. Through additional courses, I learned a number of development practices and built applications using NodeJS, Angular, React and more.

I built this site to showcase some of these projects, and to highlight some of the skills I have aquired and tools I have become familiar with throughout my time as a Web Developer.
